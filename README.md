# Data Science & AI Summer Training 🚀

Welcome to my central repository tracking my 70-hour Data Science Summer Training journey from Ducat (School of AI). 

As an engineering student, I built this repository to document my transition from foundational programming and object-oriented logic to advanced machine learning and data analytics. This space contains all my learning materials, Jupyter notebooks, lab exercises, and projects covering the end-to-end data science lifecycle.

## 📌 Course Overview
This curriculum spans 70+ hours of theory and hands-on lab sessions, encompassing 12 core modules and 5+ live industry projects. The goal is to build portfolio-ready skills, from raw data collection to production-level model deployment.

## 📚 Curriculum & Repository Structure

### Phase 1: Foundations & Data Handling
* **Module 1: Python for Data Science** - Fundamentals, OOP, NumPy, and Pandas.
* **Module 2: Mathematics & Statistics Essentials** - Descriptive/inferential statistics, probability, and linear algebra.
* **Module 3: Data Wrangling & EDA** - Data cleaning, feature engineering, and automated EDA.
* **Module 4: Data Visualisation** - Matplotlib, Seaborn, Plotly, and Power BI/Tableau dashboards.

### Phase 2: Machine Learning Core
* **Module 5: Supervised Machine Learning** - Regression, classification algorithms, and hyperparameter tuning.
* **Module 6: Unsupervised Machine Learning** - Clustering (K-Means, DBSCAN), dimensionality reduction (PCA), and anomaly detection.
* **Module 7: Feature Engineering & Model Optimisation** - Advanced feature selection, handling imbalanced data, and AutoML.

### Phase 3: Advanced AI & Deployment
* **Module 8: Deep Learning & Neural Networks** - TensorFlow/Keras, CNNs for computer vision, and RNNs/Transformers for sequences.
* **Module 9: Natural Language Processing (NLP)** - Text representation, contextual embeddings (BERT), and LLM basics.
* **Module 10: Big Data & Cloud Computing** - Apache Spark, AWS/Azure platforms, and advanced SQL.
* **Module 11: ML Model Deployment & MLOps** - Docker, FastAPI, MLflow, CI/CD with GitHub Actions.
* **Module 12: Capstone Projects** - Real-world implementations and case studies.

## 🛠️ Tech Stack & Tools Used
* **Languages:** Python 3.x, SQL, Bash/Shell
* **Data Processing & Viz:** Pandas, NumPy, Matplotlib, Seaborn, Plotly, Power BI
* **Machine Learning:** Scikit-learn, XGBoost, LightGBM
* **Deep Learning & NLP:** TensorFlow 2.x, Keras, spaCy, Hugging Face
* **Deployment & MLOps:** Flask, FastAPI, Docker, MLflow, GitHub Actions

## 📂 How to Navigate
Each folder in this repository corresponds to a specific module. Inside, you will find:
1. `notebooks/`: Jupyter notebooks with theory, code alongs, and my personal notes.
2. `labs/`: Completed lab exercises (e.g., full EDA pipelines, A/B testing analysis).
3. `projects/`: Standalone directories for the 5+ live industry projects and my final capstone.
