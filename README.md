# Data Science & AI Summer Training 🚀

Welcome to my central repository tracking my 70-hour Data Science Summer Training journey at Ducat (School of AI)[cite: 1]. 

As an engineering student, I built this repository to document my transition from foundational programming and object-oriented logic to advanced machine learning and data analytics. This space contains all my learning materials, Jupyter notebooks, lab exercises, and projects covering the end-to-end data science lifecycle[cite: 1].

## 📌 Course Overview
This curriculum spans 70+ hours of theory and hands-on lab sessions, encompassing 12 core modules and 5+ live industry projects[cite: 1]. The goal is to build portfolio-ready skills, from raw data collection to production-level model deployment[cite: 1].

## 📚 Curriculum & Repository Structure

### Phase 1: Foundations & Data Handling
* **Module 1: Python for Data Science** - Fundamentals, OOP, NumPy, and Pandas[cite: 1].
* **Module 2: Mathematics & Statistics Essentials** - Descriptive/inferential statistics, probability, and linear algebra[cite: 1].
* **Module 3: Data Wrangling & EDA** - Data cleaning, feature engineering, and automated EDA[cite: 1].
* **Module 4: Data Visualisation** - Matplotlib, Seaborn, Plotly, and Power BI/Tableau dashboards[cite: 1].

### Phase 2: Machine Learning Core
* **Module 5: Supervised Machine Learning** - Regression, classification algorithms, and hyperparameter tuning[cite: 1].
* **Module 6: Unsupervised Machine Learning** - Clustering (K-Means, DBSCAN), dimensionality reduction (PCA), and anomaly detection[cite: 1].
* **Module 7: Feature Engineering & Model Optimisation** - Advanced feature selection, handling imbalanced data, and AutoML[cite: 1].

### Phase 3: Advanced AI & Deployment
* **Module 8: Deep Learning & Neural Networks** - TensorFlow/Keras, CNNs for computer vision, and RNNs/Transformers for sequences[cite: 1].
* **Module 9: Natural Language Processing (NLP)** - Text representation, contextual embeddings (BERT), and LLM basics[cite: 1].
* **Module 10: Big Data & Cloud Computing** - Apache Spark, AWS/Azure platforms, and advanced SQL[cite: 1].
* **Module 11: ML Model Deployment & MLOps** - Docker, FastAPI, MLflow, CI/CD with GitHub Actions[cite: 1].
* **Module 12: Capstone Projects** - Real-world implementations and case studies[cite: 1].

## 🛠️ Tech Stack & Tools Used
* **Languages:** Python 3.x, SQL, Bash/Shell[cite: 1]
* **Data Processing & Viz:** Pandas, NumPy, Matplotlib, Seaborn, Plotly, Power BI[cite: 1]
* **Machine Learning:** Scikit-learn, XGBoost, LightGBM[cite: 1]
* **Deep Learning & NLP:** TensorFlow 2.x, Keras, spaCy, Hugging Face[cite: 1]
* **Deployment & MLOps:** Flask, FastAPI, Docker, MLflow, GitHub Actions[cite: 1]

## 📂 How to Navigate
Each folder in this repository corresponds to a specific module. Inside, you will find:
1. `notebooks/`: Jupyter notebooks with theory, code alongs, and my personal notes.
2. `labs/`: Completed lab exercises (e.g., full EDA pipelines, A/B testing analysis)[cite: 1].
3. `projects/`: Standalone directories for the 5+ live industry projects and my final capstone[cite: 1].
